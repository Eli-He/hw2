### how to run  
$ python triclassify.py /full/path/to/training.txt /full/path/to/testing.txt  
however, put data and triclassify.py into the same directory.  
Then run the following:  
$ python triclassify.py training.txt testing.txt  

output.txt already provided. If you prefer to regenerate one, put run.py, triclassify.py and data into same folder, then run:  
$ python run.py
